Interactive examples of plate-based data visualization for IgG results with corresponding microscopy images

The graphs show the positive ratio (Y-axis on the left) for IgG with each S/N/R/M antigen for each patient sample. Regions of the graph can be zoomed in, and the individual data points can be hovered to view 1) the image for a particular Ig/Ag and 2) the positive ratios for the particular antigen with different Igs. The page includes separate html graphs for IgG with each antigen (S/N/R/M) and the imaged 384-well plate (4 plates for each protein).
